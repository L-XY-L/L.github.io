About This Site

The list lists papers related to the topic of homomorphic encryption posted on IACR's eprint archive since January 2016.

Homomorphic encryption is a cryptographic technique based on the computational complexity theory of mathematical problems. Processing homomorphic encrypted data results in an output that is decrypted, and the result is the same as if the original unencrypted data had been processed in the same way.

I keep the list as a way for me to stay somewhat up to date with new developments in this area, and to have an easy way to find papers. The list is generated manually by going over the eprint RSS feed a few times a month. If you feel like I missed a paper let me know, and I will consider adding it to the list.

The site uses the Skeleton CSS boiler plate framework. It is hosted on GitHub and the source can be found here.
